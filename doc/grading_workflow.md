# Grading Workflow

## 1. Preparation & Setting Bobot (Admin/Akademik/Kaprodi)

1. **Master Skala Nilai**: Admin setting Skala Nilai (A, B, C...) per Prodi.
2. **Setting Komponen Penilaian**:
    - Admin/Akademik membuka menu **Data Kelas** atau **Mata Kuliah**.
    - Menentukan komponen penilaian (e.g., Tugas 20%, UTS 30%, UAS 50%) untuk setiap Mata Kuliah/Kelas.
    - PENTING: Total bobot wajib 100%.

## 2. Akses Penilaian (Dosen)

1. Dosen login dan masuk ke **Jadwal & Agenda** atau **Kelas Saya**.
2. Pilih Kelas yang diampu.
3. Masuk ke menu **Penilaian**.

## 3. Input & Submit Nilai (Dosen)

1. Masuk ke tab **Input Nilai**.
2. Dosen input nilai sesuai komponen.
3. Setelah selesai, klik **Submit Nilai**.
    - Status berubah menjadi `submitted`.
    - Nilai terkunci (Dosen tidak bisa edit lagi).
    - Nilai BELUM muncul di KHS Mahasiswa.

## 4. Verifikasi & Publikasi (Staff Prodi)

1. Staff Prodi memantau status penilaian per kelas.
2. Jika status `submitted`, Prodi melakukan verifikasi (cek kesesuaian/distribusi nilai).
3. Klik **Publish Nilai**.
    - Status berubah menjadi `published`.
    - Nilai masuk ke KHS Mahasiswa dan Transkrip.

## 5. Review Nilai (Mahasiswa)

1. Mahasiswa login.
2. Buka menu **KHS (Kartu Hasil Studi)**.
3. Lihat Nilai Akhir untuk MK tersebut.
